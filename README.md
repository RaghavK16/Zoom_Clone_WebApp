# Zoom_Clone_WebApp

![Clone_Pics](https://github.com/raghavk16/Zoom_Clone_WebApp/blob/master/zoom_clone_ss.png?raw=true)

* Simplistic Zoom Clone Web-App using NodeJS.

## Running the project

1. Install all dependencies.
```
npm i
```
2. Start the server.
```
npm start
```
3. Open localhost on port 3030.
```
localhost:3030/
```
